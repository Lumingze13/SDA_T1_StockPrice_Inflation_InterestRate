QUICKSTART INSTRUCTIONS
-----------------------

1. Ensure Python 3 and Jupyter Notebook are installed.

2. Install required packages:
   pip install pandas numpy scipy matplotlib openpyxl

3. Place the following data files in the same directory:
   - S&P_500_1982-2023.csv
   - US_Monthly_Inflation_Rate_1982-2023.xlsx
   - US_Monthly_Real_10Y_Interest_Rate_1982-2023.csv

4. Launch Jupyter Notebook:
   jupyter notebook

5. Open the "project.ipynb" file in your browser from the Jupyter interface.

6. Run all cells in the notebook to:
   - Load and preprocess the data
   - Perform statistical analysis (correlation, regression, cross-correlation)
   - Generate plots and view results directly in the notebook.
